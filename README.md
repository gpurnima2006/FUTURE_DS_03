# Marketing Funnel & Conversion Performance Analysis

## Project Overview

This project analyzes customer conversion performance using the Bank Marketing Dataset.

The objective is to understand how customers move through the marketing funnel and identify opportunities to improve conversion rates.

---

## Tools Used

* Python
* Pandas
* Matplotlib
* Jupyter Notebook

---

## Dataset

Bank Marketing Dataset

Total Records: 11,162

Features: 17

---

## Data Preparation

* Loaded dataset using Pandas
* Checked missing values
* Performed exploratory data analysis
* Calculated conversion metrics

---

## Key Findings

### Total Leads

11,162

### Converted Customers

5,289

### Conversion Rate

47.38%

### Insights

* Nearly half of the customers successfully subscribed to deposits.
* Conversion performance is relatively strong.
* Different job categories show different conversion behaviors.
* Marketing efforts can be optimized by targeting high-converting customer groups.

---

---

## Recommendations

1. Focus marketing efforts on high-converting customer segments.
2. Improve engagement strategies for low-converting groups.
3. Personalize campaign communication.
4. Increase follow-up activities for potential customers.
5. Optimize campaign timing and channel selection.

The analysis successfully identified customer conversion performance and provided actionable recommendations to improve marketing effectiveness.
